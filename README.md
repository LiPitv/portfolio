# 🌐 Portfólio Pessoal

Este repositório contém o código do meu **portfólio pessoal**, desenvolvido para apresentar os meus projetos, habilidades e informações profissionais na área de tecnologia e desenvolvimento web.

O objetivo deste portfólio é mostrar os trabalhos que desenvolvi, as tecnologias que utilizo e facilitar o contacto para oportunidades profissionais.

---

## 📌 Sobre o Projeto

O portfólio foi criado para funcionar como uma **plataforma online de apresentação profissional**, onde visitantes podem conhecer mais sobre mim, visualizar projetos realizados e entrar em contacto.

O site possui um design moderno, responsivo e organizado para proporcionar uma boa experiência ao utilizador.

---

## 🚀 Tecnologias Utilizadas

- **HTML5** – Estrutura das páginas  
- **CSS3** – Estilização e layout do site  
- **JavaScript** – Funcionalidades interativas  
- **Git & GitHub** – Controlo de versão
